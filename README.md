### :guide_dog: Installation Guide

A step by step series of examples that tell you how to get a development env running

In your cmd:

```
git clone https://github.com/RugvedB/LetsDraw/
```

Then,

```
cd LetsDraw
npm install
npm start
```

You are done with the setup now!
